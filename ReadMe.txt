Instructions for running the program:

1. Navigate to the folder containing the project files 
2. Type in "make" and hit the ENTER key
3. To run the program, type in "./plagiarismCatcher [path/to/files] 6 200"
   Hopefully you have the subdirectory containing all of the text files that 
   are to be checked in the SAME directory as the program files
   If this is not the case, please do so. for the [path/to/files] enter in the 
   name of the subdirectory containing the text files that will be checked by
   the program.
4. Hit ENTER to see which files are similar in content.

Notes:

The program is capable of running the small and medium inputs, but cannot perform 
a plagiarism check for the big input (table gets filled up).
